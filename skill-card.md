## Description
Manage GitHub Projects v2 boards using the `gh project` CLI — listing, creating, editing, and deleting projects, fields, and items. Covers the full `gh project` subcommand surface with GraphQL fallbacks for advanced mutations.

## Use Case
Agents and engineers who need to interact with GitHub Projects v2 from the command line or in automation pipelines (CI, scheduled tasks, agentic workflows).

## Requirements
- `gh` CLI installed and authenticated
- `project` scope on the token (`gh auth refresh -s project`)

## Skill Output
**Output Type(s):** Shell commands, JSON, Guidance
**Output Format:** Markdown with inline bash code blocks
